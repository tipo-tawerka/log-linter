# log-linter

Анализатор Go-проектов, проверяющий лог-сообщения на соответствие правилам стиля.

## Сборка

```bash
make build
```

Бинарный файл появится в `./bin/log-linter`.

## Запуск

```bash
# Проверить пакеты в текущем модуле
./bin/log-linter ./...

# Применить все предложенные автоисправления
./bin/log-linter -fix ./...

# Включить только определённые правила
./bin/log-linter -rules lowercase,english ./...

# Пропускать *_test.go файлы
./bin/log-linter -skiptests ./...

# Добавить чувствительные слова через запятую
./bin/log-linter -words pin,cvv,ssn ./...

# Добавить чувствительные слова из файла (по одному на строку, # — комментарии)
./bin/log-linter -wordsfile ./banned.txt ./...
```

## Примеры нарушений
